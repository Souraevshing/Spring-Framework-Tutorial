# Inserting list inside array

## Summary
- We can insert list inside `<array>` using `<list>` tag.
- Use `value` nested inside array tag.
- The return type of list inserted is of type `java.util.ArrayList`.

## Syntax

`

    <property name="marks">
        <list>
            <value>50</value>
            <value>60</value>
            <value>70</value>
        </list>
    </property>

`
